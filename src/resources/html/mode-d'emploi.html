Vous être en présence de l'outil Obviews.
Ce module sert d'interface graphique à l'utilisation du logiciel de calcul du temps maximum OTAWA. Utilisation :

Sélectionner le répertoire de travail :
Veuillez commencer par indiquer le chemin vers le répertoire de travail contenant le code source à analyser en l'écrivant dans le champ prévu à cet effet. N'oubliez pas ensuite de cliquer sur le bouton pour valider. Normalement, le chemin vers votre répertoire de travail dans la partie "infos" devrait s'actualiser.

Sélectionner le type de coloration :
Vous pouvez sélectionner grace à deux boutons si vous voulez afficher le code source ou les graphes en fonction du temps "time" ou du nombre d'itérations de chaque bloc "count".

Afficher le code source :
Cliquez sur le bouton "voir code" pour afficher la liste des fichiers C.
Cliquez ensuite sur l'un des liens pour afficher le code source du fichier C.

Afficher les graphes CFG :
Cliquez sur le bouton "voir graphe" pour afficher le graphe CFG dans un nouvel onglet.
Vous pouvez déplacer le graphe en maintenant le curseur de la souris et en déplaçant ensuite le cursuer.
Vous pouvez zoomer/dezoomer le graphe en plaçant le curseur de la souris au dessus du graphe et en faisant tourner la roulette de la souris.
Vous pouvez cliquez sur certaines parties du graphe pour accéder au sous-graphe correspondant et qui s'affichera dans un nouvel onglet.
Vous pouvez aussi directement cliquer sur un des liens en haut du graphe menant directement au graphe voulu sans avoir à passer par tous les graphes intermédiaires.

Déconnexion :
Vous pouvez stoper le serveur en cliquant sur le bouton "déconnexion".
